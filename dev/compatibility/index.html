Compatibility

Julia

  • Julia 1.10+ required by package compatibility bounds.

Python

  • System Python 3.10+ recommended.
  • sympy is required when using the SymPy backend.

Environment variables

  • JULIA_CONDAPKG_BACKEND=Null to disable Conda.
  • JULIA_PYTHONCALL_EXE=/usr/local/bin/python3 to select the system Python.

Notes

  • For homogeneous numeric or symbolic matrices, use ordinary Julia matrix literals.
  • For heterogeneous symbolic/numeric matrices that fail during construction, use mixed_matrix or @mixed_matrix to bypass matrix-literal promotion.
